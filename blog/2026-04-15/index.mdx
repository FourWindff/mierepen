Grids are invisible scaffolding that shapes how we process visual information. Understanding the mental models behind grid systems allows designers to create layouts that feel intuitive rather than arbitrary.

## The Cognitive Grid

Before computers, humans organized information spatially. Books had margins. Newspapers had columns. Cities had blocks. The grid is not a design invention — it is a cognitive primitive:

- **Predictability** — once a pattern is established, the brain expects it to continue
- **Chunking** — grids group information into manageable units
- **Scanning** — regular structure enables efficient visual search

## Grid as Constraint and Freedom

A common misconception is that grids limit creativity. In practice, constraints free cognitive resources:

```
Without grid: Where should this go? Does it align? Is it balanced?
With grid:    Which cells should this occupy?
```

The grid removes low-level positioning decisions, allowing focus on higher-level composition.

## Responsive Mental Models

As layouts adapt to different screen sizes, the underlying mental model must remain consistent:

1. **Desktop** — 12-column grid, generous whitespace
2. **Tablet** — 8-column grid, reduced margins
3. **Mobile** — 4-column grid, stacked content

The column count changes, but the principle of regular division persists. Users may not perceive the grid directly, but they sense when it is violated.

## Breaking the Grid

Deliberate grid violations create emphasis. An element that breaks the established rhythm draws attention precisely because it defies expectation. But the violation only works when the grid is strong enough to be recognizable:

> "You cannot break a grid that does not exist."

The most impactful layouts establish a rigorous grid, then strategically depart from it.
