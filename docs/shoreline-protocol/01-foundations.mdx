# Map The Coastal Baseline

Before data starts moving, the team needs a common map for stations, sectors, and handoff rules.

> Stable naming beats clever naming when the feed becomes operational.

Build the baseline around a small vocabulary:

1. `station`
2. `sector`
3. `handoff`
