# 文档加载器

文档加载器（Document Loaders）是 RAG 流程的起点，负责从各种来源读取原始内容并转为 LangChain 的 `Document` 对象。

## Document 对象

```python
from langchain_core.documents import Document

doc = Document(
    page_content="这是文档的文本内容...",
    metadata={
        "source": "path/to/file.pdf",
        "page": 1,
        "author": "张三"
    }
)
```

每个 `Document` 包含：
- **`page_content`** — 文档的文本内容（字符串）
- **`metadata`** — 任意键值对，用于检索过滤和溯源

## 本地文件加载

### 文本文件

```python
from langchain_community.document_loaders import TextLoader

loader = TextLoader("docs/article.txt", encoding="utf-8")
docs = loader.load()

print(f"加载了 {len(docs)} 个文档")
print(f"内容长度: {len(docs[0].page_content)} 字符")
print(f"元数据: {docs[0].metadata}")
```

### PDF

```python
from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("docs/report.pdf")
docs = loader.load()

# PyPDFLoader 会自动按页分割，每页一个 Document
for doc in docs:
    print(f"页 {doc.metadata['page']}: {len(doc.page_content)} 字符")

# 如果需要更精确的解析（保留表格结构），可以用 Unstructured
from langchain_community.document_loaders import UnstructuredPDFLoader
loader = UnstructuredPDFLoader("docs/report.pdf", mode="elements")
docs = loader.load()
# 每个元素（标题、段落、表格）都是独立的 Document
```

### Markdown

```python
from langchain_community.document_loaders import UnstructuredMarkdownLoader

loader = UnstructuredMarkdownLoader("docs/guide.md")
docs = loader.load()
```

### 批量加载目录

```python
from langchain_community.document_loaders import DirectoryLoader

loader = DirectoryLoader(
    "docs/",                          # 目录路径
    glob="**/*.md",                   # 文件匹配模式
    loader_cls=UnstructuredMarkdownLoader,
    show_progress=True,               # 显示进度条
    use_multithreading=True,          # 多线程加载
)
docs = loader.load()
```

## 网络加载

### 网页

```python
from langchain_community.document_loaders import WebBaseLoader

# 单页
loader = WebBaseLoader("https://python.langchain.com/docs/concepts/")
docs = loader.load()

# 多页
urls = [
    "https://python.langchain.com/docs/concepts/",
    "https://python.langchain.com/docs/tutorials/",
]
loader = WebBaseLoader(urls)
docs = loader.load()

# 自定义请求头（模拟浏览器）
loader.requests_kwargs = {
    "headers": {
        "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"
    }
}
```

### Arxiv 论文

```python
from langchain_community.document_loaders import ArxivLoader

loader = ArxivLoader(
    query="retrieval augmented generation",
    load_max_docs=5,     # 最多加载 5 篇
)
docs = loader.load()
```

### YouTube 字幕

```python
from langchain_community.document_loaders import YoutubeLoader

loader = YoutubeLoader.from_youtube_url(
    "https://youtube.com/watch?v=xxxxx",
    add_video_info=True,  # 包含标题、作者等元数据
)
docs = loader.load()
```

## 数据库加载

### 从 SQL 数据库

```python
from langchain_community.document_loaders import SQLDatabaseLoader
from langchain_community.utilities import SQLDatabase

db = SQLDatabase.from_uri("sqlite:///company.db")

loader = SQLDatabaseLoader(
    db=db,
    query="SELECT title, content FROM articles WHERE category = 'tech'"
)
docs = loader.load()
```

## 自定义加载器

如果内置加载器不满足需求，可以继承 `BaseLoader` 自定义：

```python
from langchain_core.document_loaders import BaseLoader
from langchain_core.documents import Document
from typing import Iterator

class CustomLogLoader(BaseLoader):
    """加载结构化日志文件的自定义加载器。"""

    def __init__(self, file_path: str, level_filter: str = "ERROR"):
        self.file_path = file_path
        self.level_filter = level_filter

    def lazy_load(self) -> Iterator[Document]:
        """惰性加载：逐行读取，内存友好。"""
        import json

        with open(self.file_path, "r", encoding="utf-8") as f:
            for line in f:
                log_entry = json.loads(line.strip())

                if log_entry.get("level") == self.level_filter:
                    yield Document(
                        page_content=log_entry["message"],
                        metadata={
                            "timestamp": log_entry["timestamp"],
                            "level": log_entry["level"],
                            "service": log_entry.get("service", "unknown"),
                            "source": self.file_path,
                        }
                    )

    def load(self) -> list[Document]:
        """立即加载所有文档。"""
        return list(self.lazy_load())


# 使用
loader = CustomLogLoader("logs/app.log", level_filter="ERROR")
error_docs = loader.load()
```

## 加载后处理

加载完成后，通常需要对 `Document` 列表做预处理：

```python
def preprocess_docs(docs: list[Document]) -> list[Document]:
    """文档预处理流水线。"""
    processed = []
    for doc in docs:
        # 清洗内容
        content = doc.page_content
        content = content.replace('\t', ' ')     # 制表符转空格
        content = ' '.join(content.split())      # 去除多余空白

        # 过滤空文档
        if len(content.strip()) < 10:
            continue

        processed.append(Document(
            page_content=content,
            metadata=doc.metadata
        ))
    return processed
```
